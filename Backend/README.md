# cvify-ai-platform


CVify AI 🚀

CVify AI is a full-stack GenAI-powered platform that analyzes resumes and provides actionable feedback to improve ATS compatibility and hiring chances.

⚡ FEATURES

🔍 Resume Analysis — Extract and evaluate resume content
🤖 AI Feedback — Generate improvement suggestions using LLMs
📊 ATS Scoring — Simulate Applicant Tracking System scoring
🔐 Authentication — Secure user login (JWT-based)
📄 File Upload — Resume parsing (PDF/Text)
